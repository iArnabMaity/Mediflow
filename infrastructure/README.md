# Infrastructure as Code

This directory contains infrastructure and deployment configurations.

## Contents

- **Kubernetes**: Kubernetes manifests and configurations
- **Terraform**: Terraform configurations for cloud infrastructure
- **Monitoring**: Monitoring and logging setup
- **Security**: Security policies and configurations

## Deployment

To deploy MediFlow to production:

1. Configure your cloud provider credentials
2. Review and update configuration files
3. Run deployment commands from each subdirectory
